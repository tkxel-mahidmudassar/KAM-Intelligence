# components/layout

App shell components that frame every page.

| File | What it is |
|------|-----------|
| `Sidebar.tsx` | Left nav with route links, account list shortcut, collapse toggle |
| `Topbar.tsx` | Top bar with greeting, role switcher, live sync indicator, notifications |
| `RoleSwitcher.tsx` | KAM / Manager / Executive toggle (POC auth stand-in) |
| `NavItem.tsx` | Single sidebar navigation link |
| `NotificationBell.tsx` | Unread count badge + dropdown |
| `LiveSyncIndicator.tsx` | Pulsing dot showing data freshness |
