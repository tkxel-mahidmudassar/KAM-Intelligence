# components/home

Components used exclusively on the Home daily command center page.

| File | What it is |
|------|-----------|
| `SnapshotCards.tsx` | Row of 6 summary cards (Healthy / At Risk / Critical / Open Tasks / At-Risk MRR / Renewals in 90 Days) |
| `AccountSnapshot.tsx` | Full-width sortable account table with RAG, KAM Score, MRR, next action |
| `AccountSnapshotRow.tsx` | Single account row with deep-link click behaviour |
| `CriticalActionsToday.tsx` | Full-width prioritised action queue across all accounts |
| `ActionRow.tsx` | Single action item with priority, due date, source trigger, CTA |
| `TodaysBrief.tsx` | Portfolio-level AI narrative card |
| `LiveSignals.tsx` | Feed of meaningful changes since last login |
| `SignalItem.tsx` | Single live signal entry |
