# components/ai-pulse

Portfolio-level AI health/risk/growth narrative page.

| File | What it is |
|------|-----------|
| `PulseNarrative.tsx` | Full AI-generated portfolio narrative with sources |
| `PulseRiskList.tsx` | Accounts flagged as at-risk or critical with brief reason |
| `PulseGrowthList.tsx` | Accounts with whitespace/growth opportunity signals |
| `PulseRefreshButton.tsx` | Triggers new LLM pulse generation |
| `PulseMetadata.tsx` | Shows model used, generated timestamp, confidence |
