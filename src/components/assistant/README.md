# components/assistant

KAM Assistant — RAG-backed chat interface scoped to an account.

| File | What it is |
|------|-----------|
| `AssistantPanel.tsx` | Chat panel with message history and input |
| `AssistantMessage.tsx` | Single message bubble (user or AI) with source citations |
| `AssistantInput.tsx` | Prompt input with send button |
| `SourceCitation.tsx` | Inline source reference chip with tooltip |
