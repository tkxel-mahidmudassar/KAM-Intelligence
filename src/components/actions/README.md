# components/actions

Action queue and workflow transition UI.

| File | What it is |
|------|-----------|
| `ActionCard.tsx` | Full action card with priority, due date, source trigger, status, CTA |
| `ActionQueue.tsx` | Ordered list of actions for an account or across portfolio |
| `ActionStatusBadge.tsx` | Pending / In Progress / Done / Escalated status indicator |
| `LaunchPlanModal.tsx` | Confirmation modal for launching a recovery/growth plan |
| `ActionAuditTrail.tsx` | Timeline of state transitions with actor and timestamp |
